# facaccimo

A static web app for easy management of cubari.moe sources. 

https://stirante.com/facaccimo/index.html

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
